GBM Intelligent MDT

BioPlexMDT

An intelligent multidisciplinary decision-support platform for glioblastoma.

Connect clinical information, imaging, pathology, and omics data into one intelligent GBM analysis workflow.

BioPlexMDT supports clinical case entry, MRI / DICOM imaging input, pathology TIFF immunomap simulation, single-cell omics paths, and local knowledge index retrieval for GBM MDT scenarios.

The system organizes the analysis through automated tool workflows and separates the final output into MDT recommendations, patient-friendly explanations, and professional answers.

Workflow

From GBM case data to structured MDT output

The workflow focuses on case preparation, tool execution, evidence integration, and structured result presentation.

1 Case and Question Entry

Enter the chief complaint, history, imaging findings, pathology results, molecular tests, and clinical questions.

2 Multimodal Data Input

Provide MRI, DICOM, pathology TIFF, h5ad, or 10x data paths for backend tool execution.

3 Automated Tool Workflow

Run imaging, pathology, omics, retrieval, and reasoning tools with live status updates.

4 Layered Result Output

Generate MDT recommendations, patient-friendly explanations, professional answers, and simulated immunomaps.
