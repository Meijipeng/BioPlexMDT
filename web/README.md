# BioPlexMDT Frontend

This folder contains the static frontend pages for BioPlexMDT.

## Structure

- `index.html`: project home page
- `work.html`: case submission workspace
- `result.html`: result feedback page
- `contact.html`: contact page
- `css/styles.css`: site-wide visual styles
- `js/app.js`: canvas visuals, API calls, and form interactions
- `assets/`: static image assets

## Backend Port

The frontend calls:

```text
http://localhost:8000
```

## Usage

Open `index.html` in a browser, or serve the `web/` folder through the FastAPI backend.
