# GigaTIME Driver

This folder only contains BioPlexMDT driver code for calling an external GigaTIME installation.

Place the official GigaTIME project and its model weights outside this repository, then set:

```text
BIOPLEX_GIGATIME_EXTERNAL_SCRIPT=
```

Default BioPlexMDT entry:

```text
tools/GigaTIME/scripts/custom_inference.py
```
