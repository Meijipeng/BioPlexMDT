# ROAM Driver

This folder only contains BioPlexMDT driver code for calling an external ROAM installation.

Place ROAM and its model weights outside this repository, then set:

```text
BIOPLEX_ROAM_EXTERNAL_ROOT=
BIOPLEX_ROAM_ROOT=tools/ROAM
```

Driver entry:

```text
tools/ROAM/run_roam_driver.py
```
